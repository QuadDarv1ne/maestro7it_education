# Хранимые процедуры

В этой папке находятся файлы с хранимыми процедурами для часто используемых операций в базе данных зоопарка.

## Содержимое

- `stored_procedures.sql` - хранимые процедуры для часто используемых операций

## Описание

Файл включает в себя следующие процедуры:
- `AddNewAnimal` - добавление нового животного
- `AddFeeding` - добавление записи о кормлении
- `AddMedicalRecord` - добавление медицинской записи
- `GetHealthReport` - получение отчета о здоровье животных
- `GetVaccinationDueList` - получение списка животных, нуждающихся в вакцинации
