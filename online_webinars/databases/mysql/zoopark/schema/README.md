# Схема базы данных

В этой папке находятся файлы, отвечающие за создание структуры базы данных зоопарка.

## Содержимое

- `01_database_schema.sql` - создание базы данных, таблиц и связей между ними

## Описание

Файл включает в себя:
- Создание базы данных `animals_db`
- Определение всех таблиц (species, animals, enclosures, employees, feedings, medical_records, vaccinations, diets, animal_enclosures, employee_enclosures)
- Определение первичных и внешних ключей
- Создание начальных индексов для оптимизации производительности
