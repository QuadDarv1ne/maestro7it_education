# Вспомогательные системы

В этой папке находятся файлы различных вспомогательных систем для базы данных зоопарка.

## Содержимое

- `analytics_system.sql` - система аналитики и отчетности
- `auth_system.sql` - система аутентификации и авторизации пользователей
- `data_archiving_system.sql` - система архивирования старых данных
- `examples.sql` - примеры использования хранимых процедур
- `logging_system.sql` - система логирования действий пользователей
- `monitoring_system.sql` - система мониторинга и алертинга
- `tests.sql` - автоматизированные тесты для проверки корректности работы процедур

## Описание

Каждый файл содержит соответствующую систему:
- `analytics_system.sql` - предоставляет аналитическую информацию и отчеты
- `auth_system.sql` - управляет доступом пользователей к системе
- `data_archiving_system.sql` - архивирует старые данные для оптимизации производительности
- `examples.sql` - содержит примеры использования хранимых процедур
- `logging_system.sql` - отслеживает действия пользователей в системе
- `monitoring_system.sql` - осуществляет мониторинг состояния базы данных
- `tests.sql` - автоматизированные тесты для проверки корректности работы процедур
