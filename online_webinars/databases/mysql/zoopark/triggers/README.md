# Триггеры и ограничения

В этой папке находятся файлы с триггерами и ограничениями для обеспечения целостности данных в базе данных зоопарка.

## Содержимое

- `triggers_constraints.sql` - триггеры и ограничения для обеспечения целостности данных

## Описание

Файл включает в себя:
- Триггеры для проверки корректности вводимых данных
- Ограничения (constraints) для проверки значений
- Дополнительные поля для отслеживания состояния животных
- Триггеры для автоматического обновления связанных данных
