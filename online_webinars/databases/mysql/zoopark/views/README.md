# Представления (Views)

В этой папке находятся файлы с представлениями для удобного доступа к данным базы данных зоопарка.

## Содержимое

- `03_create_views.sql` - создание представлений (views) для удобного доступа к данным
- `04_view_queries.sql` - примеры запросов к представлениям

## Описание

Файл `03_create_views.sql` включает в себя:
- `animals_full_info` - полная информация о животных с деталями вида и вольера
- `todays_feeding_schedule` - расписание кормлений на сегодня
- `animal_relocation_history` - история перемещений животных между вольерами
- `medical_history` - медицинская история животных
- `enclosure_statistics` - статистика по вольерам
- `vaccination_due` - животные, нуждающиеся в вакцинации
- `animal_diet_schedule` - расписание питания по животным
- `family_tree` - родственные связи животных
- `employee_responsibilities` - обязанности сотрудников
- `daily_health_report` - ежедневный отчет по здоровью

Файл `04_view_queries.sql` содержит примеры использования этих представлений.
