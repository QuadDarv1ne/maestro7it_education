# Данные базы данных

В этой папке находятся файлы с начальными данными для базы данных зоопарка.

## Содержимое

- `02_sample_data.sql` - наполнение базы данных начальными данными

## Описание

Файл включает в себя:
- Вставку данных в таблицу видов (species)
- Вставку данных в таблицу вольеров (enclosures)
- Вставку данных в таблицу сотрудников (employees)
- Вставку данных в таблицу животных (animals)
- Вставку данных в таблицу кормлений (feedings)
- Вставку данных в таблицу медицинских записей (medical_records)
- Вставку данных в таблицу вакцинаций (vaccinations)
- Вставку данных в таблицу диет (diets)
- Вставку данных в таблицу истории размещения животных (animal_enclosures)
- Вставку данных в таблицу назначений сотрудников (employee_enclosures)
- Проверку количества записей в каждой таблице
