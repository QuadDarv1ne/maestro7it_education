## –û–ø–µ—Ä–∞—Ç–æ—Ä—ã

### –ê—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –ù–∞–∑–≤–∞–Ω–∏–µ | –ü—Ä–∏–º–µ—Ä | –†–µ–∑—É–ª—å—Ç–∞—Ç |
|----------|----------|--------|-----------|
| `+` | –°–ª–æ–∂–µ–Ω–∏–µ | `5 + 3` | `8` |
| `-` | –í—ã—á–∏—Ç–∞–Ω–∏–µ | `10 - 4` | `6` |
| `*` | –£–º–Ω–æ–∂–µ–Ω–∏–µ | `3 * 7` | `21` |
| `/` | –î–µ–ª–µ–Ω–∏–µ | `20 / 4` | `5` |
| `%` | –û—Å—Ç–∞—Ç–æ–∫ –æ—Ç –¥–µ–ª–µ–Ω–∏—è | `10 % 3` | `1` |
| `++` | –ò–Ω–∫—Ä–µ–º–µ–Ω—Ç | `x++` | `x = x + 1` |
| `--` | –î–µ–∫—Ä–µ–º–µ–Ω—Ç | `x--` | `x = x - 1` |

```cpp

```

### –û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –ó–Ω–∞—á–µ–Ω–∏–µ | –ü—Ä–∏–º–µ—Ä | –†–µ–∑—É–ª—å—Ç–∞—Ç |
|----------|----------|--------|-----------|
| `==` | –†–∞–≤–Ω–æ | `5 == 5` | `true` |
| `!=` | –ù–µ —Ä–∞–≤–Ω–æ | `5 != 3` | `true` |
| `>` | –ë–æ–ª—å—à–µ | `10 > 5` | `true` |
| `<` | –ú–µ–Ω—å—à–µ | `3 < 8` | `true` |
| `>=` | –ë–æ–ª—å—à–µ –∏–ª–∏ —Ä–∞–≤–Ω–æ | `7 >= 7` | `true` |
| `<=` | –ú–µ–Ω—å—à–µ –∏–ª–∏ —Ä–∞–≤–Ω–æ | `4 <= 6` | `true` |


### –û–ø–µ—Ä–∞—Ç–æ—Ä—ã –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –≠–∫–≤–∏–≤–∞–ª–µ–Ω—Ç | –ü—Ä–∏–º–µ—Ä |
