# Директория библиотек PCMetrics

Эта директория содержит поддиректории для различных библиотек графических процессоров, используемых в проекте PCMetrics.

## Структура директории:

```
libs/
├── nvml/     # Библиотека управления NVIDIA (NVIDIA Management Library)
├── adl/      # Библиотека отображения AMD (AMD Display Library)
└── intel/    # Анализаторы производительности графики Intel (Intel Graphics Performance Analyzers)
```

## Общая информация:

Каждая поддиректория предназначена для хранения соответствующих библиотек и файлов разработчиков для поддержки мониторинга GPU различных производителей:

1. **nvml/** - для мониторинга графических карт NVIDIA
2. **adl/** - для мониторинга графических карт AMD
3. **intel/** - для мониторинга интегрированной графики Intel

## Инструкции по использованию:

1. Перейдите в нужную поддиректорию (nvml, adl или intel)
2. Прочитайте файл README.md в каждой директории для получения подробных инструкций по установке
3. Скачайте соответствующие библиотеки с официальных сайтов производителей
4. Поместите файлы библиотек в соответствующие директории
5. Соберите проект PCMetrics с соответствующими флагами CMake:
   - Для NVIDIA: `-DENABLE_NVML=ON`
   - Для AMD: `-DENABLE_ADL=ON`
   - Для Intel: `-DENABLE_INTEL_GPA=ON`

## Важные замечания:

- Все библиотеки являются собственностью соответствующих производителей
- Убедитесь, что соблюдаете лицензионные соглашения производителей
- Для работы библиотек требуются соответствующие драйверы оборудования
- Библиотеки могут быть недоступны для скачивания без регистрации на сайтах производителей
- Версии библиотек должны быть совместимы с вашим оборудованием и операционной системой

## Поддержка:

Если у вас возникли проблемы с установкой или использованием библиотек, обратитесь к документации в каждой поддиректории или к основной документации проекта PCMetrics.
