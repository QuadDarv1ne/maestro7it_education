### –°–æ–∑–¥–∞–Ω–∏–µ DataFrame

`–û—Å–Ω–æ–≤–Ω–æ–π –æ–±—ä–µ–∫—Ç Pandas` ‚Äî —ç—Ç–æ `DataFrame`, –¥–≤—É–º–µ—Ä–Ω–∞—è —Ç–∞–±–ª–∏—Ü–∞ —Å –¥–∞–Ω–Ω—ã–º–∏, –ø–æ—Ö–æ–∂–∞—è –Ω–∞ —ç–ª–µ–∫—Ç—Ä–æ–Ω–Ω—É—é —Ç–∞–±–ª–∏—Ü—É.

–í—ã –º–æ–∂–µ—Ç–µ —Å–æ–∑–¥–∞—Ç—å `DataFrame` –∏–∑ —Å–ª–æ–≤–∞—Ä–µ–π, —Å–ø–∏—Å–∫–æ–≤ –∏–ª–∏ –¥—Ä—É–≥–∏—Ö —Å—Ç—Ä—É–∫—Ç—É—Ä –¥–∞–Ω–Ω—ã—Ö.

```python

        '–ì–æ—Ä–æ–¥': ['–ú–æ—Å–∫–≤–∞', '–°–∞–Ω–∫—Ç-–ü–µ—Ç–µ—Ä–±—É—Ä–≥', '–ö–∞–∑–∞–Ω—å']}
```

**–ù–∞–ø—Ä–∏–º–µ—Ä, —á—Ç–æ–±—ã –∑–∞–≥—Ä—É–∑–∏—Ç—å `CSV-—Ñ–∞–π–ª`:**

```python
# –°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –ø–æ —É–±—ã–≤–∞–Ω–∏—é –≤–æ–∑—Ä–∞—Å—Ç–∞
```

#### –†–∞–±–æ—Ç–∞ —Å –ø—Ä–æ–ø—É—â–µ–Ω–Ω—ã–º–∏ –¥–∞–Ω–Ω—ã–º–∏

**Pandas –ø–æ–º–æ–≥–∞–µ—Ç –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—Ç—å –ø—Ä–æ–ø—É—â–µ–Ω–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è (NaN):**

1. `df.isnull()` ‚Äî –ø—Ä–æ–≤–µ—Ä—è–µ—Ç, –≥–¥–µ –µ—Å—Ç—å –ø—Ä–æ–ø—É—Å–∫–∏.  

2. `df.fillna(value)` ‚Äî –∑–∞–ø–æ–ª–Ω—è–µ—Ç –ø—Ä–æ–ø—É—Å–∫–∏ –∑–∞–¥–∞–Ω–Ω—ã–º –∑–Ω–∞—á–µ–Ω–∏–µ–º.

–ú–µ—Ç–æ–¥ `groupby` –ø–æ–∑–≤–æ–ª—è–µ—Ç –≥—Ä—É–ø–ø–∏—Ä–æ–≤–∞—Ç—å –¥–∞–Ω–Ω—ã–µ –∏ –ø—Ä–∏–º–µ–Ω—è—Ç—å –∫ –Ω–∏–º —Ñ—É–Ω–∫—Ü–∏–∏, —Ç–∞–∫–∏–µ –∫–∞–∫ `sum`, `mean` –∏–ª–∏ `count`

#### –°–ª–∏—è–Ω–∏–µ –∏ –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ –¥–∞–Ω–Ω—ã—Ö
