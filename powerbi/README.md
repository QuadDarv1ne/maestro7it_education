# Power BI Обучающий Проект

## Описание

Комплексный обучающий материал по Power BI с практическими примерами, шаблонами и кодом.

## Структура проекта

```
powerbi/
├── powerbi_manual.md          # Основное руководство
├── powerbi_advanced.md        # Продвинутые техники
└── examples/                  # Практические примеры
    ├── power_query/          # Примеры Power Query
    │   ├── data_cleaning.pq
    │   └── advanced_techniques.pq
    ├── dax/                  # Примеры DAX
    │   ├── basic_formulas.dax
    │   └── advanced_analytics.dax
    ├── visualizations/       # Шаблоны визуализаций
    │   └── dashboard_templates.json
    ├── templates/            # Шаблоны отчетов
    │   └── report_templates.json
    └── data_samples/         # Образцы данных
        └── README.md
```

## Начало работы

1. Установите Power BI Desktop
2. Изучите `powerbi_manual.md`
3. Практикуйтесь с примерами из папки `examples/`
4. Используйте шаблоны для создания собственных отчетов

## Основные разделы

### Теория
- Введение в Power BI
- Установка и настройка
- Основные компоненты
- Моделирование данных
- Язык DAX

### Практика
- Power Query преобразования
- Создание визуализаций
- Продвинутая аналитика
- Безопасность данных

### Примеры
- Очистка данных
- Финансовый анализ
- HR-аналитика
- Маркетинговая аналитика
- Логистика

## Требования

- Power BI Desktop (последняя версия)
- Windows 10 или выше
- 4 ГБ RAM минимум
