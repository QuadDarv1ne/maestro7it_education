Подробное руководство по установке DuckDB на Windows (с Python)

1. Проверка наличия Python
--------------------------
Перед установкой DuckDB убедитесь, что на вашем компьютере установлен Python версии 3.7 или выше.

Откройте командную строку (cmd) или PowerShell и выполните:
    python --version

Если команда не распознана:
- Скачайте Python с официального сайта: https://www.python.org/downloads/windows/
- Во время установки ОБЯЗАТЕЛЬНО отметьте галочку "Add Python to PATH".

2. Установка DuckDB через pip
-----------------------------
DuckDB предоставляет официальный Python-пакет, который устанавливается одной командой.

Откройте командную строку (от имени пользователя, не обязательно администратора) и выполните:

    pip install duckdb

Эта команда автоматически скачает и установит последнюю стабильную версию DuckDB для вашей системы [[1], [3]].

> Примечание: Для большинства пользователей Windows дополнительные зависимости не требуются. Однако в редких случаях (например, при сборке из исходников) может понадобиться Microsoft Visual C++ Redistributable [[6]].

3. Проверка установки
---------------------
Создайте тестовый Python-файл (например, test_duckdb.py) со следующим содержимым:

    import duckdb
    result = duckdb.sql("SELECT 'DuckDB успешно установлен!' AS message;")
    print(result)

Запустите его из командной строки:
    python test_duckdb.py

Если вы видите сообщение "DuckDB успешно установлен!" — всё работает корректно.

4. Дополнительные возможности (опционально)
-------------------------------------------
- DuckDB поддерживает работу с CSV, Parquet, JSON без дополнительных библиотек.
- Для работы с Pandas установите его отдельно: pip install pandas
- Для расширенной функциональности (например, HTTPFS для загрузки файлов из интернета) можно установить расширения через SQL:
      INSTALL httpfs;
      LOAD httpfs;

5. Устранение распространённых проблем
--------------------------------------
- Если pip выдаёт ошибку "command not found", убедитесь, что Python и pip добавлены в PATH.
- Если возникает ошибка при импорте, попробуйте обновить pip: python -m pip install --upgrade pip
- Для пользователей Anaconda/Miniconda также поддерживается установка через conda:
      conda install python-duckdb -c conda-forge

Заключение
----------
DuckDB — это встраиваемая аналитическая СУБД без внешних зависимостей.
Установка на Windows занимает менее 10 секунд и не требует настройки сервера [[1], [3], [5]].

Официальная документация: https://duckdb.org/docs/
