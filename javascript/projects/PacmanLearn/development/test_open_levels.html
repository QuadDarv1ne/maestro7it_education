Тест открытия уровней в Pacman

Проверка функциональности

Этот тест проверяет, что все уровни теперь открыты для выбора

Нажмите кнопку выше для запуска теста

Инструкция по использованию

Теперь все уровни доступны без необходимости проходить предыдущие уровни:

  • Запустите игру Pacman
  • Перейдите в меню выбора уровней
  • Все 10 уровней должны быть доступны для выбора
  • Выберите любой уровень и начните играть
