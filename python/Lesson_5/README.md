# Работа с коллекциями данных в Python: Списки, кортежи, множества и словари

Python предоставляет несколько встроенных типов коллекций для работы с набором данных.

**Рассмотрим основные из них:** `списки`, `кортежи`, `множества` и `словари`.

### 1. Списки (list)

**Списки** — это изменяемые, упорядоченные коллекции данных. Они могут хранить элементы любых типов.

**Основные операции со списками:**

- **Создание списка:**
```python
my_list = [1, 2, 3, 4]
```

- **Добавление элемента в конец:**
```python
my_list.append(5)
```

- **Вставка элемента по индексу:**
```python
my_list.insert(2, 'a')  # Вставляет 'a' на позицию с индексом 2
```

- **Удаление элемента:**
```python
my_list.remove(2)  # Удаляет первое вхождение элемента '2'
```

- **Получение длины списка:**
```python
len(my_list)  # Возвращает количество элементов
```

- **Индексация и срезы:**
```python
print(my_list[0])  # Первый элемент
print(my_list[1:3])  # Элементы с индекса 1 до 3 (не включая)
```

- **Объединение списков:**
```python
new_list = my_list + [6, 7, 8]
```

### 2. Кортежи (tuple)

**Кортежи** — это неизменяемые, упорядоченные коллекции данных.

Как и списки, они могут хранить элементы любых типов, но их нельзя изменять после создания.

**Основные операции с кортежами:**

- **Создание кортежа:**
```python
my_tuple = (1, 2, 3)
```

- **Индексация и срезы работают аналогично спискам:**
```python
print(my_tuple[0])  # Первый элемент
print(my_tuple[1:3])  # Элементы с индекса 1 до 3
```

- **Использование кортежей для множественного присваивания:**
```python
a, b, c = my_tuple  # a = 1, b = 2, c = 3
```

### 3. Множества (set)

**Множества** — это неупорядоченные коллекции уникальных элементов.

Они используются для хранения и операций с уникальными значениями.

**Основные операции с множествами:**

- **Создание множества:**
```python
my_set = {1, 2, 3, 4}
```

- **Добавление элемента:**
```python
my_set.add(5)
```

- **Удаление элемента:**
```python
my_set.remove(3)
```

- **Объединение множеств:**
```python
set1 = {1, 2, 3}
set2 = {3, 4, 5}
union_set = set1.union(set2)  # {1, 2, 3, 4, 5}
```

- **Пересечение множеств:**
```python
intersection_set = set1.intersection(set2)  # {3}
```

- **Разность множеств:**
```python
difference_set = set1.difference(set2)  # {1, 2}
```

### 4. Словари (dict)

**Словари** — это неупорядоченные коллекции пар `"ключ-значение"`.

Ключи должны быть уникальными и неизменяемыми, а значения могут быть любыми.

**Основные операции со словарями:**

- **Создание словаря:**
```python
my_dict = {'name': 'Alice', 'age': 25}
```

- **Доступ к значению по ключу:**
```python
print(my_dict['name'])  # 'Alice'
```

- **Добавление или обновление элемента:**
```python
my_dict['job'] = 'Engineer'  # Добавление нового ключа
my_dict['age'] = 26  # Обновление существующего ключа
```

- **Удаление элемента:**
```python
my_dict.pop('age')  # Удаляет ключ 'age' и возвращает его значение
```

- **Получение всех ключей или значений:**
```python
keys = my_dict.keys()  # Получение всех ключей
values = my_dict.values()  # Получение всех значений
```

- **Перебор ключей и значений:**
```python
for key, value in my_dict.items():
    print(f'{key}: {value}')
```

### Подведение итогов

Списки используются для работы с изменяемыми последовательностями, **кортежи** — для неизменяемых последовательностей, **множества** — для работы с уникальными элементами, а **словари** — для хранения данных в формате **{ключ-значение}**.
