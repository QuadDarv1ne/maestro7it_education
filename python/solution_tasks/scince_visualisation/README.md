# Улучшенная система академических визуализаций

## Описание
Эта система генерирует 8 академических графиков для статьи «Эстетическая новизна в музыке, сгенерированной ИИ: критерии, измерения и философские импликации».

## Улучшения в версии 4.1.0

### Основные улучшения:
- ✅ Добавлена проверка зависимостей перед запуском
- ✅ Улучшена обработка ошибок для внешних библиотек
- ✅ Добавлена резервная реализация для statsmodels
- ✅ Улучшена совместимость с различными версиями matplotlib
- ✅ Добавлены дополнительные проверки для методов Axes

### Графики:
1. **График 1**: Культурные профили эстетической новизны (радар-диаграмма)
2. **График 2**: Распределение человеческой и ИИ-генерированной музыки в латентном пространстве (t-SNE визуализация)
3. **График 3**: Зависимость перцептивной оценки от вычислительной новизны
4. **График 4**: Онтологическая карта философских концептов эстетической новизны
5. **График 5**: Динамика новизны во времени
6. **График 6**: Сравнение по жанрам
7. **График 7**: Анализ корреляций между метриками
8. **График 8**: Визуализации доверительных интервалов

## Требования
- Python 3.7+
- numpy
- matplotlib
- scipy
- statsmodels (опционально)

## Установка зависимостей
```bash
pip install numpy matplotlib scipy
```

## Запуск
```bash
python google_colab_graphics_fixed.py
```

## Результаты
Графики будут сохранены в директории `novelty_visualizations/figures/` в форматах PNG, PDF и SVG.
