О проекте

Цель проекта

Помочь школьникам определить свои профессиональные склонности и выбрать подходящую образовательную траекторию через научно обоснованные методики тестирования.

Научная основа

Проект основан на проверенных психологических методиках, разработанных отечественными и зарубежными учеными в области профессиональной ориентации.

Используемые технологии

Python Flask

Веб-фреймворк

SQLite

База данных

Bootstrap 5

Фронтенд фреймворк

Для кого этот проект

  • Школьники 9-11 классов - для выбора профиля обучения
  • Родители - для понимания склонностей ребенка
  • Педагоги - для профессионального консультирования
  • Психологи - как инструмент диагностики
