Научная основа проекта

Проект основан на проверенных психологических методиках, разработанных ведущими отечественными и зарубежными учеными в области профессиональной ориентации.

Методика Климова

Дифференциально-диагностический опросник

Разработана Е.А. Климовым для определения склонности к одной из пяти типовых сфер труда. Помогает выявить профессиональные предпочтения по системе "человек-природа-техника-знак-художественный образ".

Методика Холланда

RIASEC модель

Разработана Джоном Холландом для определения профессиональных предпочтений через оценку шести типов личности. Позволяет соотнести тип личности с подходящей профессиональной средой.
