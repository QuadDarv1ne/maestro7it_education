API Documentation

Test Results API

GET /api/v1/test-results

Получить список результатов тестов

Parameters:
  • page (optional) - Номер страницы
  • per_page (optional) - Количество элементов на странице
Response:
{
  "results": [...],
  "pagination": {...}
}
GET /api/v1/test-results/{id}

Получить конкретный результат теста

Response:
{
  "id": 1,
  "user_id": 1,
  "test_type": "klimov",
  "results": {...},
  "created_at": "2024-01-01T00:00:00"
}
