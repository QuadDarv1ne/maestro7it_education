ООП (Объектно-Ориентированное Программирование) — это парадигма (стиль) программирования, в которой программа представляется как набор взаимодействующих объектов, каждый из которых является экземпляром определенного класса.

> Можно представить это как создание виртуальных "предметов" со своими свойствами и возможностями, которые взаимодействуют друг с другом.
