# План тестирования HR-системы

## 1. Тест-кейсы для CRUD операций

### TC001: Создание сотрудника
**Шаги:**
1. Перейти в раздел "Сотрудники"
2. Нажать кнопку "Добавить сотрудника"
3. Заполнить все обязательные поля
4. Нажать "Сохранить"

**Ожидаемый результат:** Сотрудник успешно создан и отображается в списке

**Фактический результат:** Сотрудник успешно создан

**Статус:** Пройден

### TC002: Редактирование сотрудника
**Шаги:**
1. Перейти в раздел "Сотрудники"
