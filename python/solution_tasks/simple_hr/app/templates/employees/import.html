Загрузка файла CSV
Файл должен содержать следующие столбцы: full_name, email, employee_id, hire_date, department, position
Формат файла CSV

Пример содержимого файла CSV:

full_name,email,employee_id,hire_date,department,position
Иванов Иван Иванович,ivanov@example.com,EMP001,2023-01-15,Отдел разработки,Программист
Петрова Мария Сергеевна,petrova@example.com,EMP002,2023-02-01,Отдел маркетинга,Маркетолог

Важно:

  • Дата приема должна быть в формате YYYY-MM-DD
  • Поля full_name, email, employee_id обязательны для заполнения
  • Если подразделение или должность не существуют, они будут созданы автоматически
  • Сотрудники с уже существующим email или табельным номером будут пропущены
