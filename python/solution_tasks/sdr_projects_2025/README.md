## –£–ø—Ä–∞–≤–ª–µ–Ω–∏–µ —Ä–∞–¥–∏–æ—á–∞—Å—Ç–æ—Ç–∞–º–∏ –≤ Python

–î–ª—è —Ä–∞–±–æ—Ç—ã —Å —Ä–∞–¥–∏–æ—á–∞—Å—Ç–æ—Ç–∞–º–∏ –≤ Python –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è `SDR` (`Software-Defined Radio`) —É—Å—Ç—Ä–æ–π—Å—Ç–≤–∞, —Ç–∞–∫–∏–µ –∫–∞–∫ `RTL-SDR`, `HackRF`, `USRP` –∏–ª–∏ `LimeSDR`

1. `–ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ RTL-SDR (–¥–µ—à—ë–≤—ã–π –≤–∞—Ä–∏–∞–Ω—Ç)`

`RTL-SDR` —Ä–∞–±–æ—Ç–∞–µ—Ç –≤ –¥–∏–∞–ø–∞–∑–æ–Ω–µ 24‚Äì1766 –ú–ì—Ü (–∑–∞–≤–∏—Å–∏—Ç –æ—Ç –º–æ–¥–µ–ª–∏).

**–ü—Ä–∏–º–µ—Ä –ø—Ä–æ–≥—Ä–∞–º–º—ã –¥–ª—è —Å–∫–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏—è FM-—Ä–∞–¥–∏–æ (88‚Äì108 –ú–ì—Ü):**

```python

# –ù–∞—Å—Ç—Ä–æ–π–∫–∞ –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤
sdr.sample_rate = 2.4e6  # –ß–∞—Å—Ç–æ—Ç–∞ –¥–∏—Å–∫—Ä–µ—Ç–∏–∑–∞—Ü–∏–∏

sdr.gain = 'auto'         # –ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∞—è –Ω–∞—Å—Ç—Ä–æ–π–∫–∞ —É—Å–∏–ª–µ–Ω–∏—è

# –ß—Ç–µ–Ω–∏–µ –¥–∞–Ω–Ω—ã—Ö

# –ü–æ—Å—Ç—Ä–æ–µ–Ω–∏–µ —Å–ø–µ–∫—Ç—Ä–∞

plt.ylabel('–ú–æ—â–Ω–æ—Å—Ç—å (–¥–ë)')
plt.show()

```

**–ü—Ä–∏–º–µ—Ä –ø–µ—Ä–µ–¥–∞—á–∏ —Å–∏–Ω—É—Å–æ–∏–¥–∞–ª—å–Ω–æ–≥–æ —Å–∏–≥–Ω–∞–ª–∞:**
