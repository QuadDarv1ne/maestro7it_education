# Система управления задачами на Reflex

Это веб-приложение для управления задачами, созданное с использованием фреймворка Reflex. Приложение позволяет создавать, отслеживать и управлять задачами с различными приоритетами, статусами и сроками выполнения.

## Особенности

- Добавление новых задач с названием, описанием, приоритетом и сроком выполнения
- Назначение задач конкретным исполнителям
- Изменение статуса задач (Ожидание, В процессе, Выполнено, Отменено)
- Фильтрация задач по статусу и приоритету
- Отображение статистики по задачам
- Интуитивно понятный веб-интерфейс

## Требования

- Python 3.8 или выше
- Reflex

## Установка

1. Убедитесь, что у вас установлен Python 3.8+
2. Установите Reflex:
   ```
   pip install reflex
   ```
3. Клонируйте этот репозиторий
4. Перейдите в директорию проекта:
   ```
   cd task_management_system
   ```
5. Запустите приложение:
   ```
   reflex run
   ```

## Использование

1. Откройте веб-браузер и перейдите по адресу http://localhost:8000
2. Используйте форму "Добавить новую задачу" для создания задач
3. Управляйте задачами с помощью кнопок в карточках задач
4. Фильтруйте задачи по статусу и приоритету
5. Просматривайте статистику выполнения задач в реальном времени

## Структура проекта

- `task_management_system.py` - основной файл приложения
- `rxconfig.py` - конфигурационный файл Reflex
- `.web/` - внутренняя директория для сборки веб-приложения
- `assets/` - директория для хранения статических ресурсов
