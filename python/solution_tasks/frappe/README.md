# Frappe Project Management System

> Комплексное решение для управления проектами на базе фреймворка `Frappe`

## Обзор

`Frappe` — это мощный фреймворк с открытым исходным кодом для создания ERP-систем и бизнес-приложений. Этот проект представляет собой готовое решение для управления проектами, которое позволяет эффективно отслеживать задачи, сроки, ресурсы и бюджеты проектов.

## Возможности

- **Управление проектами** — Создание и отслеживание проектов с возможностью указания сроков, статусов и прогресса
- **Управление задачами** — Назначение задач участникам команды с приоритетами и сроками выполнения
- **Управление ресурсами** — Отслеживание человеческих ресурсов, оборудования и материалов
- **Управление бюджетом** — Контроль за планируемыми и фактическими расходами с анализом отклонений
- **Отчетность** — Генерация отчетов по проектам и производительности
- **Информационная панель** — Персонализированный дашборд для каждого пользователя

## Архитектура решения

### Основные типы документов (DocTypes):

1. **Проект** (`Project`)
   - Заголовок и описание проекта
   - Статус (Открыт, В работе, Завершен, Отменен)
   - Сроки начала и завершения
   - Прогресс выполнения
   - Менеджер проекта
   - Общий бюджет

2. **Задача** (`Task`)
   - Название и описание задачи
   - Привязка к проекту
   - Исполнитель
   - Приоритет (Низкий, Средний, Высокий)
   - Сроки выполнения
   - Плановые и фактические часы

3. **Ресурс** (`Resource`)
   - Имя ресурса
   - Тип (Человек, Оборудование, Материал)
   - Почасовая ставка
   - Описание

4. **Бюджет** (`Budget`)
   - Привязка к проекту
   - Категория бюджета
   - Планируемая сумма
   - Фактическая сумма
   - Отклонение и процент отклонения

5. **Таймшит** (`Project Timesheet`)
   - Привязка к проекту и задаче
   - Сотрудник
   - Дата работы
   - Затраченные часы
   - Почасовая ставка и общая сумма

6. **Веха проекта** (`Project Milestone`)
   - Привязка к проекту и задаче
   - Название вехи
   - Статус (Ожидает, В процессе, Завершено)
   - Плановая и фактическая даты

7. **Шкала времени проекта** (`Project Timeline`)
   - Привязка к проекту и задаче
   - Название события
   - Тип (Веха, Этап, Задача, Событие)
   - Даты начала и окончания
   - Ответственное лицо

8. **Аналитика проекта** (`Project Analytics`)
   - Статистика по проектам, задачам, ресурсам и бюджетам
   - Дашборд с ключевыми метриками
   - Детализированные отчеты

## Отчеты

- **Сводка проектов** — Обзор всех проектов с ключевыми метриками
- **Сводка таймшитов** — Отчет по затраченному времени сотрудников
- **Аналитика проектов** — Комплексная статистика по проектам, задачам, ресурсам и бюджетам
- Фильтры по проектам, сотрудникам и датам
- Интерактивные диаграммы и графики

## Установка

1. Убедитесь, что у вас установлен Frappe Bench

   ```bash
   # Установка bench
   pip install frappe-bench
   ```

2. Клонируйте репозиторий в папку приложений

   ```bash
   cd ~/frappe-bench/apps
   git clone https://github.com/your-repo/project-management.git
   cd project-management
   pip install -e .
   ```

3. Установите приложение в сайт

   ```bash
   bench --site your-site.local install-app project_management
   ```

4. Перезапустите сервер

   ```bash
   bench start
   ```
