# Система управления проектами для Frappe

Комплексное приложение управления проектами, построенное на фреймворке Frappe, которое позволяет отслеживать задачи, сроки, ресурсы и бюджеты.

## Возможности

- **Управление проектами**: Создание и управление проектами с заголовками, описаниями, статусами и отслеживанием прогресса
- **Управление задачами**: Назначение задач членам команды с приоритетами, сроками и учетом времени
- **Управление ресурсами**: Отслеживание человеческих ресурсов, оборудования и материалов с учетом стоимости
- **Управление бюджетом**: Мониторинг планируемых и фактических затрат с анализом отклонений
- **Отчетность**: Просмотр сводок проектов и аналитики
- **Информационная панель**: Персонализированная панель с отображением ваших проектов и задач

## Установка

1. Скопируйте этот репозиторий в директорию приложений Frappe
2. Установите приложение в ваш сайт Frappe:
   ```bash
   bench get-app project_management /path/to/project_management_system
   bench install-app project_management
   ```
3. Перезапустите сервер bench

## Использование

После установки приложение "Управление проектами" будет доступно в вашем рабочем столе Frappe. Вы можете:

- Создавать новые проекты из типа документа "Проект"
- Добавлять задачи к проектам с конкретными исполнителями
- Отслеживать использование ресурсов
- Мониторить бюджеты и расходы
- Просматривать отчеты и информационные панели

## Типы документов

- **Проект**: Основная сущность проекта с отслеживанием прогресса
- **Задача**: Отдельные задачи, привязанные к проектам
- **Ресурс**: Человеческие ресурсы/оборудование с тарифами
- **Бюджет**: Финансовый учет с анализом отклонений

## Отчеты

- **Сводка проекта**: Обзор всех проектов с ключевыми показателями

## Пользовательские страницы

- **Панель проекта**: Персонализированная панель для пользователей
