# Полный справочник открытых астрономических данных и ресурсов

Здесь собраны ссылки на основные открытые архивы, каталоги, программные инструменты и поисковые системы для работы с данными космических миссий и наземных обсерваторий.

Источники сгруппированы по тематике для удобства навигации.

---

## 🔭 Миссии NASA и общие архивы

### Портал открытой науки NASA

**[science.data.nasa.gov](https://science.data.nasa.gov/)** — главная входная точка во все научные данные NASA. Здесь собраны репозитории по всем дисциплинам: астрофизике, гелиофизике, планетологии и наукам о Земле.

**Ключевые разделы:**

- **Science Discovery Engine** — интегрированный поиск по данным, публикациям, коду и инструментам NASA.
- **Science Explorer (SciX)** — цифровая библиотека для поиска научных публикаций и ПО по астрономии, физике Земли и планетологии.

### MAST (Mikulski Archive for Space Telescopes)

**[archive.stsci.edu](https://archive.stsci.edu)** — основной архив данных космических телескопов, включая:
- **Хаббл (HST)** — все наблюдения с 1990 года.
- **Джеймс Уэбб (JWST)** — полные массивы данных, включая недавний релиз 1.5 ТБ по проекту COSMOS-Web.
- **TESS, Kepler, K2** — данные по экзопланетам.
- **GALEX, XMM-Newton** (частично) и другие.
- **WISE** — каталог всего неба в инфракрасном диапазоне.

**Удобные интерфейсы MAST:**

- **[MAST Portal]
(https://mast.stsci.edu/portal/Mashup/Clients/Mast/Portal.html)** — визуальный поиск и скачивание.
- **[Astroquery](https://astroquery.readthedocs.io/)** (модуль Python `astroquery.mast`) — программный доступ.

## 🌍 Европейские и международные ресурсы

### ESA Datalabs

**[datalabs.esa.int](https://datalabs.esa.int)** — совместная научная платформа Европейского космического агентства. Позволяет работать с большими данными (например, каталоги GAIA) прямо в облаке, используя Jupyter notebooks. Уже доступны данные Solar Orbiter и JWST.

### ESASky

**[sky.esa.int](https://sky.esa.int)** — интерактивная карта неба от ESA. Позволяет визуально исследовать данные множества миссий, включая Hubble, JWST, XMM-Newton, Gaia и Planck. Простой и наглядный инструмент для начала.

### CADC (Canadian Astronomy Data Centre)

**[www.cadc-ccda.hia-iha.nrc-cnrc.gc.ca](https://www.cadc-ccda.hia-iha.nrc-cnrc.gc.ca)** — канадский центр с доступом к данным Hubble, JWST, CFHT и других обсерваторий.

### Страсбургский центр астрономических данных (CDS)

**[cds.unistra.fr](https://cds.unistra.fr)** — одна из старейших и крупнейших мировых баз астрономических данных. Содержит каталоги звезд, галактик и внегалактических объектов.

## 🪐 Планетология и Солнечная система

### PDS (Planetary Data System) NASA

**[pds.nasa.gov](https://pds.nasa.gov)** — основной архив всех данных планетарных миссий NASA. Данные разделены по тематическим узлам (nodes).

**Ключевые узлы:**

- **PDS Small Bodies Node (SBND)** — данные по астероидам, кометам и малым телам (например, каталог наблюдений New Horizons PEPSSI).
- **PDS Geosciences Node** — данные по геологии планет и спутников.
- **PDS Atmospheres Node** — данные по атмосферам планет.

### JPL Solar System Dynamics

**[ssd.jpl.nasa.gov](https://ssd.jpl.nasa.gov)** — здесь находятся:
- **HORIZONS System** — главный сервис для расчета эфемерид планет, комет, астероидов и космических аппаратов.
- **Small-Body Database** — орбиты и физические параметры малых тел.
- **NEO Program** — данные по околоземным объектам, включая датасеты по потенциально опасным астероидам.

### Minor Planet Center (MPC)

**[minorplanetcenter.net](https://minorplanetcenter.net)** — международный центр сбора данных о малых планетах и кометах. Все наблюдения астероидов стекаются сюда [citation:9].

### USGS Astrogeology

**[astrogeology.usgs.gov](https://astrogeology.usgs.gov)** — данные по планетной картографии. Здесь можно найти карты Луны, Марса и других тел.
- **AstroLink Archives** — коллекция исторических материалов и публикаций USGS по планетологии.

## ✨ Рентгеновская и высокоэнергетическая астрономия

### HEASARC (High Energy Astrophysics Science Archive Research Center)

**[heasarc.gsfc.nasa.gov](https://heasarc.gsfc.nasa.gov)** — архив данных по рентгеновской и гамма-астрономии.

**Основной ресурс для миссий:**

- **Chandra X-ray Observatory** — рентгеновские данные высокого разрешения.
- **XMM-Newton, NuSTAR, Swift, Fermi** и другие.

**Важный ресурс:** [Chandra Source Catalog (CSC 2.1)](https://cxc.harvard.edu/csc/) — каталог источников, зарегистрированных обсерваторией Chandra. Версия 2.1 (2025) содержит более **400 000 уникальных рентгеновских объектов**.

## 🔧 Инструменты для визуализации и анализа

### Aladin Lite

**[aladin.u-strasbg.fr/AladinLite/](https://aladin.u-strasbg.fr/AladinLite/)** — интерактивный атлас неба прямо в браузере. Позволяет накладывать изображения из разных каталогов и архивов.

### JHelioviewer

**[jhelioviewer.org](https://www.jhelioviewer.org)** — инструмент для визуализации данных Солнца и гелиосферы (SDO, SOHO, Solar Orbiter).

### NASA Science Cloud

Единая облачная среда NASA для ускорения научных открытий. Доступна для исследователей.

### High-End Computing

Программа доступа к суперкомпьютерам NASA для решения сложных вычислительных задач.

## 📚 Специализированные каталоги и коллекции

### Hubble Spectroscopic Legacy Archive (HSLA)

**[archive.stsci.edu/hlsp/hsla](https://archive.stsci.edu/hlsp/hsla)** — все ультрафиолетовые спектры телескопа Hubble. Обновленная версия включает автоматическую классификацию объектов.

### PEGASUS Stellar Spectra

Генератор полных спектров звезд (включая экстремальный ультрафиолет) на основе моделей.

### Opacities Database

База данных непрозрачности для изучения атмосфер экзопланет.

### COSMOS-Web Catalog (JWST)

Каталог, опубликованный в июне 2025 года. Содержит фотометрию, структурные параметры и физические характеристики для почти **800 000 галактик** по данным JWST.
