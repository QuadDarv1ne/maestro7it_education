## Что такое квантовые вычисления?

**Квантовые вычисления** — это область вычислительной техники, которая использует квантовые явления, такие как **суперпозиция** и **запутанность**, для обработки информации. В отличие от классических компьютеров, которые используют биты (0 или 1), квантовые компьютеры используют **кубиты**, которые могут находиться в состоянии 0, 1 или любого квантового состояния между ними.

Квантовые вычисления были предложены в 1980-х годах физиками, такими как **Ричард Фейнман** и **Юрий Манин**, как способ моделирования квантовых систем, которые невозможно эффективно смоделировать с помощью классических компьютеров.

### Основные характеристики квантовых вычислений:

- **Параллелизм** благодаря суперпозиции
- **Запутанность** для создания сложных корреляций
- **Интерференция** для усиления правильных ответов и подавления неправильных
- **Экспоненциальное увеличение мощности** с ростом количества кубитов

## Фундаментальные понятия квантовой физики

### Квантовые биты (кубиты)

**Классический бит может быть либо 0, либо 1. Кубит, с другой стороны, может находиться в суперпозиции этих состояний:**

$$ |\psi\rangle = \alpha|0\rangle + \beta|1\rangle $$

где α и β — комплексные числа, называемые амплитудами вероятности, такие что $ |\alpha|^2 + |\beta|^2 = 1 $.

### Суперпозиция

Суперпозиция позволяет кубиту одновременно находиться во всех своих возможных состояниях. Это означает, что n кубитов могут представлять $ 2^n $ состояний одновременно, что дает экспоненциальный выигрыш в емкости хранения по сравнению с классическими битами.

### Квантовая запутанность

Запутанность — это квантовое явление, при котором пары или группы частиц создают коррелированные состояния. Когда кубиты запутаны, измерение одного из них мгновенно влияет на другие, независимо от расстояния между ними.

### Квантовые логические вентили

`Квантовые вентили` — это унитарные преобразования, которые изменяют состояние кубитов.

**Некоторые важные квантовые вентили:**

- **X-вентиль (NOT-вентиль)**: $ \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix} $
- **H-вентиль (Адамара)**: $ \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix} $
- **CNOT-вентиль**: используется для создания запутанности

### Измерения в квантовой механике

Измерение квантового состояния приводит к "коллапсу" суперпозиции в одно из базовых состояний (0 или 1) с вероятностью, определяемой квадратом амплитуды этого состояния.

## Практическое квантовое программирование

### Установка и настройка Qiskit

`Qiskit` — это открытая платформа для квантового программирования, разработанная IBM.

#### Установка Qiskit:

```bash
pip install qiskit
pip install qiskit-aer  # Для симуляции
pip install qiskit-ibmq-provider  # Для доступа к квантовым компьютерам IBM
```

#### Простой пример квантовой схемы:

```python
from qiskit import QuantumCircuit, ClassicalRegister, QuantumRegister
from qiskit import execute, Aer
from qiskit.visualization import plot_histogram

# Создаем квантовую схему с 2 кубитами и 2 классическими битами
qc = QuantumCircuit(2, 2)

# Применяем вентиль Адамара к первому кубиту
qc.h(0)

# Применяем CNOT вентиль
qc.cx(0, 1)

# Измеряем кубиты
qc.measure([0,1], [0,1])

# Визуализируем схему
print(qc.draw())
```

### Создание квантовых схем

Квантовые схемы строятся из последовательности квантовых вентилей, которые изменяют состояние кубитов.

**Схемы могут включать:**

- Инициализацию кубитов
- Применение вентилей
- Измерения
- Условные операции

### Симуляция квантовых алгоритмов

**Qiskit предоставляет несколько симуляторов:**

- **Statevector simulator**: моделирует идеальные квантовые системы
- **Qasm simulator**: моделирует с учетом шума
- **Unitary simulator**: вычисляет унитарную матрицу всей схемы

### Работа с реальным квантовым hardware

`IBM` предоставляет доступ к реальным квантовым процессорам через их облачную платформу `IBM Quantum Experience`

1. Зарегистрироваться на сайте IBM Quantum

## Квантовые алгоритмы

### Алгоритм Шора

`Алгоритм Шора` позволяет факторизовать большие числа за полиномиальное время, что представляет серьезную угрозу для современной криптографии `RSA`

#### Основные шаги:

1. Проверить, является ли число четным или простым
2. Найти период функции $ f(x) = a^x \mod N $
3. Использовать квантовую Фурье-трансформацию для определения периода
4. Вычислить факторы N

### Алгоритм Гровера

Алгоритм Гровера позволяет искать в неупорядоченном массиве с квадратичным ускорением по сравнению с классическим поиском.

#### Основные шаги:

1. Подготовить суперпозицию всех возможных решений
2. Применить oracle для пометки правильного решения
3. Инвертировать амплитуды относительно среднего
4. Повторить шаги 2-3 примерно $ \sqrt{N} $ раз

### Квантовая Фурье-трансформация (QFT)

`QFT` является квантовой версией `дискретной Фурье-трансформации` и используется во многих квантовых алгоритмах, включая алгоритм Шора.

## Прикладные темы

### Квантовая криптография

Квантовая криптография использует принципы квантовой механики для обеспечения безопасности коммуникаций.

`Главное преимущество` — возможность обнаружения попыток перехвата.

### Квантовое распределение ключей (QKD)

QKD позволяет двум сторонам безопасно обмениваться секретным ключом, при этом любая попытка перехвата будет обнаружена.

### Протокол BB84

Один из первых и самых известных протоколов QKD, разработанный Беннетом и Брассардом в 1984 году.

**Он использует два базиса поляризации фотонов:**

- **Прямолинейный:** 0° и 90°
- **Диагональный:** 45° и 135°

### Современные криптосистемы

Квантовые вычисления ставят под угрозу многие современные криптографические системы, такие как `RSA` и `ECC`

**Поэтому разрабатываются:**

- Постквантовые криптографические алгоритмы
- Квантово-устойчивые протоколы
- Гибридные системы безопасности
