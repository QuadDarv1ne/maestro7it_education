О проекте ChessCalendar-RU

Описание проекта

ChessCalendar-RU - это веб-приложение для отслеживания шахматных турниров в России. Проект автоматически собирает информацию с официальных источников FIDE и Федерации шахмат России.

Источники данных
  • FIDE Calendar
  • Федерация шахмат России
Технологии
  • Python Flask
  • SQLite Database
  • BeautifulSoup4 (парсинг)
  • Bootstrap 5 (фронтенд)
