# ChessCalendar-RU

Веб-приложение для отслеживания шахматных турниров в России 2026 года.

## Описание

ChessCalendar-RU автоматически собирает и отображает информацию об официальных шахматных турнирах в России из источников FIDE и Федерации шахмат России.

## Функционал

- Просмотр календаря турниров
- Автоматический парсинг данных с официальных сайтов
- Фильтрация турниров по категориям
- Детальная информация о каждом турнире
- API для интеграции с другими приложениями

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd ChessCalendar-RU
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Запустите приложение:
```bash
python run.py
```

5. Откройте в браузере: http://127.0.0.1:5000

## Использование

### Основные страницы

- `/` - Главная страница с календарем турниров
- `/about` - Информация о проекте
- `/update` - Ручное обновление данных

### API endpoints

- `GET /api/tournaments` - Получить все турниры
- `GET /api/tournaments/<id>` - Получить турнир по ID
