# Адаптивный дизайн ChessCalendar-RU

## Обзор

Приложение полностью адаптировано для всех типов устройств и разрешений экрана от мобильных телефонов до 4K мониторов.

## Breakpoints (Точки останова)

### Определенные breakpoints:

| Название | Размер | Описание |
|----------|--------|----------|
| **xs** | 0-575px | Мобильные телефоны (портрет) |
| **sm** | 576-767px | Мобильные телефоны (ландшафт) |
| **md** | 768-991px | Планшеты |
| **lg** | 992-1199px | Маленькие ноутбуки |
| **xl** | 1200-1399px | Ноутбуки и десктопы |
| **xxl** | 1400-1599px | Большие десктопы |
| **fhd** | 1600-1919px | Full HD (1920x1080) |
| **2k** | 1920-2559px | 2K мониторы |
| **4k** | 2560px+ | 4K и выше |

## Адаптивная сетка карточек

### Количество колонок по разрешениям:

- **Mobile (до 575px)**: 1 колонка
- **Small tablets (576-767px)**: 2 колонки
- **Tablets (768-991px)**: 2 колонки
- **Desktop (992-1399px)**: 3 колонки
- **Large Desktop (1400-1599px)**: 3 колонки
- **Full HD (1600-1919px)**: 4 колонки
- **2K (1920-2559px)**: 4 колонки
- **4K (2560px+)**: 5 колонок

## Адаптивные размеры карточек

### Минимальная высота карточек:

- **Mobile**: 380px

- **Desktop**: 450px
- **Full HD+**: 480px

## Компоненты

### 1. Hero Section
- Адаптивные размеры заголовков
- Скрытие декоративных элементов на мобильных
- Оптимизированные отступы

### 2. Статистические карточки
- Адаптивные размеры иконок
- Оптимизированная высота
- Гибкая типографика

### 3. Фильтры
- Вертикальная компоновка на мобильных
- Полноширинные кнопки на маленьких экранах
- Оптимизированные размеры полей

### 4. Карточки турниров
- Адаптивная высота
- Оптимизированные отступы
- Гибкая компоновка контента

### 5. Навигация
- Адаптивное меню
- Мобильная нижняя навигация
- Оптимизированный поиск

### 6. Footer
- Адаптивная сетка (1/2/4 колонки)
- Оптимизированные размеры ссылок
- Гибкая компоновка

1. **adaptive-complete.css** - Основная адаптивная система
2. **tournament-cards-fix.css** - Исправления карточек
3. **visual-enhancements.css** - Визуальные эффекты

## Использование

Все адаптивные стили применяются автоматически. Для оптимальной работы убедитесь, что:

1. Все CSS файлы подключены в правильном порядке
2. Viewport meta tag установлен корректно
3. Браузер поддерживает CSS Grid и Flexbox

## Поддержка браузеров

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## Известные ограничения

- Некоторые анимации отключены на мобильных для производительности
- Glassmorphism может не работать в старых браузерах
- CSS Grid требует современный браузер
